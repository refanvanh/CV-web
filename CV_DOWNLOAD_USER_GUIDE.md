# CV Download Feature - User Guide

## Pengenalan Fitur Download CV

Fitur ini memungkinkan Anda untuk mendownload CV/Portofolio Anda dalam dua format berbeda:
- **PDF** - Format profesional untuk dokumen
- **JPG** - Format gambar untuk dibagikan di media sosial

## Cara Menggunakan

### Langkah 1: Klik Tombol Download
Cari tombol **"Download"** dengan icon download di navbar bagian atas halaman.

![Download Button Location]
Tombol berwarna cyan/turquoise terletak di antara menu navigasi dan tombol Edit.

### Langkah 2: Pilih Format
Setelah mengklik tombol download, sebuah modal akan muncul dengan dua pilihan:

**üìÑ PDF Format**
- Ideal untuk dokumen profesional
- Cocok untuk mengirim email ke recruiter atau hiring manager
- Ukuran file lebih besar karena kualitas tinggi
- Dapat dibuka di semua device dengan PDF reader

**üñºÔ∏è JPG Format**
- Ideal untuk dibagikan di media sosial (LinkedIn, Instagram, Twitter)
- Ukuran file lebih kecil
- Dapat langsung di-crop atau di-edit sebelum dibagikan
- Cocok untuk portfolio showcase

### Langkah 3: Tunggu Proses Download
Saat Anda memilih format:
1. Layar akan menampilkan pesan "Sedang mempersiapkan CV Anda..."
2. Sistem akan mengcapture seluruh konten halaman
3. File akan otomatis didownload ke folder Downloads Anda
4. Notifikasi sukses akan muncul di pojok kanan atas

## File yang Dihasilkan

### Naming Convention
File akan disimpan dengan format nama:
```
[Nama_Anda]_[Tanggal].pdf
[Nama_Anda]_[Tanggal].jpg
```

### Contoh
```
Reza_Fadjar_Nawawi_21-01-2026.pdf
Reza_Fadjar_Nawawi_21-01-2026.jpg
```

Tanggal menggunakan format Indonesia (DD-MM-YYYY)

## Tips & Trik

### Sebelum Download
1. **Review konten** - Pastikan semua informasi sudah benar dan up-to-date
2. **Check gambar** - Pastikan foto profil dan proyek sudah tampil dengan baik
3. **Scroll seluruh halaman** - Tekan Ctrl+Home untuk ke paling atas halaman
4. **Close modal lain** - Tutup modal login atau edit jika masih terbuka

### Ukuran dan Format
- **PDF** lebih cocok untuk email atau apply pekerjaan formal
- **JPG** lebih cocok untuk LinkedIn profile atau Instagram portfolio
- **Kualitas PDF** lebih terjaga daripada JPG untuk print

### Sharing
#### LinkedIn
1. Download sebagai JPG
2. Upload ke LinkedIn post atau profile picture
3. Tambah caption yang menarik

#### Email
1. Download sebagai PDF
2. Attach ke email application Anda
3. Ceritakan alasan Anda melamar

#### Website/Portfolio
1. Download sebagai JPG
2. Compress ukuran file di tool online seperti tinypng.com
3. Upload ke website atau portfolio Anda

## Troubleshooting

### ‚ùå Tombol Download tidak terlihat
**Solusi:**
- Refresh halaman (F5 atau Ctrl+R)
- Pastikan browser sudah support JavaScript
- Clear browser cache (Ctrl+Shift+Delete)

### ‚ùå Download tidak jalan
**Solusi:**
- Tunggu beberapa detik sampai proses selesai
- Cek download folder atau notification panel browser
- Coba browser lain (Chrome, Firefox, Edge)
- Disable ad blocker jika ada

### ‚ùå Gambar tidak tampil di download
**Solusi:**
- Pastikan internet connection stabil
- Coba refresh halaman terlebih dahulu
- Tunggu semua gambar loading penuh
- Coba download lagi

### ‚ùå File terlalu besar
**Solusi:**
- Format PDF biasanya lebih besar dari JPG
- Gunakan JPG untuk file lebih kecil
- Compress file di tool online setelah download
- Hapus cache browser untuk space lebih besar

### ‚ùå Notifikasi tidak muncul
**Solusi:**
- Check browser notification settings
- Cek bagian atas layar atau pojok kanan atas
- Refresh page dan coba lagi

## FAQ (Frequently Asked Questions)

**Q: Berapa lama proses download?**
A: Tergantung ukuran halaman dan kecepatan koneksi, biasanya 2-5 detik.

**Q: Bisa edit CV sebelum download?**
A: Ya, klik tombol Edit di navbar untuk mengubah konten, lalu download.

**Q: File didownload dimana?**
A: Ke folder "Downloads" default browser Anda. Bisa diubah di settings browser.

**Q: Bisa download hanya bagian tertentu?**
A: Saat ini download seluruh halaman. Untuk bagian tertentu, gunakan screenshot biasa (Print Screen).

**Q: Bisa ganti nama file saat download?**
A: File otomatis terbuat. Anda bisa rename setelah download.

**Q: Format apa yang terbaik?**
A: PDF untuk formal/email, JPG untuk social media.

**Q: Berapa ukuran file biasanya?**
A: PDF 1-3 MB, JPG 200 KB - 1 MB (tergantung gambar).
